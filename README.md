# gridsome-plugin-webpack-size

Logs your production bundle sizes in a CI friendly way.

![](./gridsome-plugin-webpack-size.png, "Screenshot showing the output of this plugin")

## Installation

`yarn add gridsome-plugin-webpack-size`

## Usage

`gridsome.config.js`
```js
module.exports = {
  plugins: [
    'gridsome-plugin-webpack-size'
  ]
}
```
